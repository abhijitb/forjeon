== Description ==

**Forjeon** is a comprehensive WordPress Gutenberg enhancement plugin that transforms your block editor experience. It provides advanced design controls, professional blocks, and a powerful toolbar interface that rival premium page builders while maintaining WordPress native compatibility.

Forjeon aims to be the **ultimate Gutenberg enhancement suite** - providing all the advanced features found in premium plugins, but with superior organization, performance, and user experience through our innovative dedicated toolbar interface.

= Key Features =

**🎨 Floating Toolbar Interface**
Access all Forjeon controls through a floating, draggable toolbar accessible via a header button. No sidebar clutter - just clean, organized design tools.

**⚡ Advanced Typography Controls**
Fine-tune letter spacing, line height, and text shadows with real-time preview. Professional typography control at your fingertips.

**📑 Professional Tabs Block**
Multi-style tabs with responsive accordion behavior and smooth animations. Choose from Default, Pills, or Underline styles.

**🔧 Developer-Friendly Architecture**
Built with modern PHP 8.2+ standards, PSR-4 autoloading, and comprehensive testing tools.

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "Forjeon"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Upload to `/wp-content/plugins/forjeon/`
3. Activate the plugin through the 'Plugins' menu in WordPress

= Getting Started =

1. **Install and activate** the Forjeon plugin
2. **Create a new post/page** and look for the **🎨 Forjeon** button in the editor header
3. **Click the Forjeon button** to open the floating toolbar
4. **Explore typography controls** - Select any text block and use the Typography tab
5. **Add professional blocks** - Insert a Tabs block from the Forjeon category

== Frequently Asked Questions ==

= What are the minimum requirements? =

* PHP 8.2 or higher
* WordPress 6.5 or higher
* Modern browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)

= Is Forjeon compatible with other plugins? =

Yes! Forjeon is designed to work seamlessly with WordPress core and follows all WordPress coding standards for maximum compatibility.

= How do I access the Forjeon toolbar? =

Look for the **🎨 Forjeon** button in your WordPress editor header (next to the settings). Click it to open the floating toolbar that you can drag anywhere in your workspace.

= Can I use Forjeon with any theme? =

Absolutely! Forjeon works with any properly coded WordPress theme as it extends the core Gutenberg editor functionality.

= Is there a pro version? =

Currently, Forjeon is completely free. We're focused on building the best possible Gutenberg enhancement experience for everyone.

= How can I contribute or report issues? =

Visit our GitHub repository or use the WordPress support forums. We welcome contributions, bug reports, and feature suggestions.
