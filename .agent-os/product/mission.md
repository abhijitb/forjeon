# Product Mission

## Pitch

**Forjeon** is a WordPress Gutenberg enhancement plugin that helps site builders, designers, and developers create visually compelling websites by providing all the advanced blocks, design controls, and styling options missing from the default Gutenberg editor.  

**Tagline:** *Forjeon — crafted blocks, limitless design.*

## Users

### Primary Customers

- **WordPress Site Builders:** Freelancers and agencies who need to deliver polished websites quickly without coding everything manually.  
- **Content Creators & Marketers:** Bloggers, marketers, and non-technical users who want more layout/design flexibility in the editor.  
- **Web Designers:** Professionals who want granular design control, animations, and typography options without relying on heavy page builders.  

### User Personas

**Freelance Site Builder** (25-40)  
- **Role:** WordPress Developer / Freelancer  
- **Context:** Building multiple client sites using block-based themes.  
- **Pain Points:** Limited Gutenberg styling, missing blocks, too many extra plugins to fill gaps.  
- **Goals:** Deliver feature-rich, design-flexible sites faster with fewer dependencies.  

**Marketing Manager** (28-45)  
- **Role:** Content & Marketing Lead  
- **Context:** Needs to publish engaging pages without developer help.  
- **Pain Points:** Gutenberg feels restrictive for layout, lacks animations/interactive blocks.  
- **Goals:** Easily create polished pages that look modern and professional.  

## The Problem

### Gutenberg Limitations

The default Gutenberg editor lacks many essential blocks and advanced design controls. This forces users to either install multiple separate plugins or switch to heavier page builders, leading to **fragmented workflows, inconsistent design, and slower site performance**.  

**Our Solution:** Forjeon provides a single, modular plugin that consolidates missing blocks, adds responsive visual controls, and extends Gutenberg with animations, icons, and custom CSS editing—all while keeping performance lightweight.  

## Differentiators

### All-in-One Enhancer

Unlike standalone block libraries or bulky page builders, Forjeon provides **a curated set of missing Gutenberg features in one package**. This results in **fewer dependencies, consistent UX, and reduced maintenance overhead**.  

### Refined UX and Stability

Unlike competitors like Editor Plus, Forjeon offers **cleaner UI grouping, better handling of theme overrides, and bug-free style persistence**. This results in **a smoother editing experience and fewer user frustrations**.  

## Key Features

### Core Features

- **Missing Blocks:** Tabs, Accordion, Icon, Counter, Progress Bar, Countdown Timer, etc. → gives users functional flexibility without third-party plugins.  
- **Advanced Visual Controls:** Responsive spacing, typography, colors, borders, shadows, visibility, and hover effects → delivers design freedom without custom CSS.  
- **Shape Dividers:** Built-in SVG dividers for Cover, Group, and Columns blocks → enables modern, visually dynamic layouts.  
- **Animations:** No-code animations like Fade, Slide, Zoom, Flip → makes content interactive and engaging.  
- **Icon Integration:** Insert icons inline with text and blocks, styled with colors and borders → improves readability and design clarity.  
- **Custom CSS Editor:** Block-specific or global CSS editing for power users → provides ultimate flexibility for designers/developers.  

### Collaboration Features

- **Configurable Feature Toggles:** Enable/disable blocks or controls per project → ensures teams only use what’s needed.  
- **Shared Standards Support:** Works seamlessly with Agent-OS `.agent-os/standards/` → helps distributed teams maintain consistent design and coding practices.
